# Data files

- `pair_catalog_auto.csv`: primer-pair catalog used by the benchmark.
- `accession_species_list.csv`: GenBank accession identifiers and species labels used in the worked example.
- `species_counts.csv`: accession counts by species.

FASTA and GenBank flatfiles are not included here; regenerate them from the accession list or provide local files in the folder structure shown in the main README.
